### 介绍

该项目是自己在完成导师的爬虫任务时想到的一个idea，界面样式参考八爪鱼PRA，功能全部由自己独立实现。项目分为web前端、chrome插件端、后端三个部分，前端负责构建可视化拖拽页面，插件端负责爬取数据，后端负责数据存储及前端交互。（目前前端界面相对完整，插件端和后端开发由于时间精力有限暂时搁浅，后续将继续完善）
